# 📖 Overview

**AegisCipher Pro** is a professional desktop application that allows users to securely encrypt and decrypt text using industry-standard cryptographic algorithms.

Designed with a modern cybersecurity-inspired interface, the application combines security, simplicity, and performance into one powerful encryption suite.

This project was developed as **Task 01** during my **Cybersecurity Internship at Pinnacle Labs Pvt Ltd.**

- 256-bit Encryption
- High Security
- Fast Performance
- Symmetric Encryption

---

## DES (Data Encryption Standard)

- 56-bit Key
- Educational Implementation
- Symmetric Encryption

---

## RSA

- 2048-bit Key Pair
- Public/Private Keys
- Asymmetric Encryption
- Secure Key Exchange

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Dhrumin1327/AegisCipher-Pro.git
```

Move into the project

```bash
cd AegisCipher-Pro
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```

1. Launch the application
2. Select Encryption Algorithm
3. Enter Plain Text
4. Generate or Load Key
5. Encrypt / Decrypt
6. Copy the Output
